Что такое Front-End?

Фронтенд-разработка —

это создание той части сайта или веб-приложения, с которой взаимодействует пользователь. Это все, что вы видите в браузере: дизайн, кнопки, тексты, изображения и анимации.

Фронтенд-разработчики используют три основные технологии: HTML для описания структуры страниц, CSS для их стилизации и JavaScript для добавления интерактивных элементов.
